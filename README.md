## Project Description
This project demonstrates basic Linux and shell scripting skills using open-source tools. It includes five scripts that perform system analysis, package checking, disk auditing, log analysis, and manifesto generation.

---

## Scripts Overview

### 1. organizer.sh
Organizes files into folders (Images, Documents, Videos), compresses them, and shows file details.

### 2. script2.sh
Checks if Python is installed and displays version information.

### 3. script3.sh
Displays directory permissions and disk usage of important system folders.

### 4. script4.sh
Analyzes a log file and counts occurrences of the keyword "error".

### 5. script5.sh
Generates a personalized open-source manifesto based on user input.
